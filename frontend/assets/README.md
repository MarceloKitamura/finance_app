# assets/

Pasta reservada para ícones, imagens e logos do app.

Hoje o app usa emojis como ícones (sem dependências). Quando quiser
ícones "de verdade", coloque os SVG/PNG aqui e referencie com caminho
relativo: `assets/logo.svg` (a partir da raiz do frontend) ou
`../assets/logo.svg` (a partir de `pages/`).
